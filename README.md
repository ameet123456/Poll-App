# Real-Time Poll App (V2)

A real-time polling application built with:
- Node.js
- Express
- Socket.IO
- React
- TailwindCSS

## Features
- Create polls
- Shareable voter & admin links
- Live voting with Socket.IO
- Admin dashboard with live results
- Stop poll feature

## Tech Stack
Frontend: React + Vite + Tailwind  
Backend: Node.js + Express + Socket.IO  

## Run Locally

### Backend
cd server  
npm install  
node index.js  

### Frontend
cd client  
npm install  
npm run dev
